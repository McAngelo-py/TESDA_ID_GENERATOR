# TESDA_ID_GENERATOR

A local-only Python desktop application for batch replacement of names and IDs in `.docx` files.

## Features
- Upload a Word (`.docx`) template.
- Input name mappings (OLD NAME = NEW NAME) via textbox or CSV file.
- Batch replace names in paragraphs and tables.
- Modern GUI using `tkinter`.
- Local-only processing (no internet connection required).

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python src/main.py
   ```
